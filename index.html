Kubtoron



Kubtoron is a sandbox turn-based game. The idea of the game is to expand your city with the money that you get from space investment. This game was developed for the TGA game talent showcase.

Our game develops on unity. In the space trading phase, we have to display candle strick. In this part, I use my candlestick chart that you can see from my free assets section. Moreover, for displaying a diagram I have to write my data convertor from CSV to my scriptable object on unity.

news management

news customization

We have a news system that will show the situations about space. News will be different every turn. For this part, I develop a news display by using a scriptable object to edit news easily. In the same way, I can custom my building by using a scriptable object.

One more step to love you


One more step to love you were developed in the Monogame framework. The story is about a freshman girl who has to fight for her love. This game is a puzzle game. Player has to force people who block the path to walk toward her love. But you need to think about how you move people because the people will return to their original position.

In our game, we have to display dialogues that are related to illustration. Because of that, we need to develop a dialogue edit system to edit or customize dialogue easy as possible. Moreover, we have our game management system that can remove or create a new object.

In level-design part, we decide to create a simple level edit that uses grid coordinates to edit people in a scene or custom level. So, if we want to develop a more advanced level-editor, we can develop from this program.

Checker Game


Checker game is a console application game that was created in C language. In this game, I work as a core programmer.

The idea for programming is to store the checkerboard in a 2-dimensional array. Next, display the array by looping the array and displaying each element on the checkerboard. Moreover, this game can detect error moves. And, players can input new moves. For this logic, we can easily make with design function that separates their job. So, We can easily input a new move and check the new move even do something more. Our Checker game also has a king pierce that can move both forward and backward. King can jump in several squares. By checker game rules we have multiple captures our game also has that too.
